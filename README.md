# SimplyAppoint

SimplyAppoint is an appointment management web application built using ASP.NET Core MVC and C#.

The project focuses on backend architecture, structured MVC design, and database integration.

## Features
- Create, edit, and delete appointments (CRUD operations)
- MVC architecture
- Database integration
- Basic validation logic

## Tech Stack
- C#
- ASP.NET Core MVC
- .NET
- SQL Server

## Project Status
Currently in development and actively being improved.

## Purpose
This project was built to strengthen backend development skills and apply object-oriented programming principles in a real-world application.
